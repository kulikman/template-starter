# Command: /project:fix-issue

Исправляет конкретный баг или проблему.

## Использование

```
/project:fix-issue "инвойс не сохраняется при нулевой сумме"
/project:fix-issue "TypeScript ошибка в src/lib/calculations.ts:42"
/project:fix-issue "RLS не работает для admin роли"
```

## Процесс

1. Воспроизведи проблему (найди код где она возникает)
2. Прочитай `.claude/memory/mistakes.md` — может уже фиксили похожее
3. Примени агента `coder` для исправления
4. Напиши тест который воспроизводит баг (red → green)
5. Зафикси в `.claude/memory/mistakes.md` что сломалось и как починили

## Output

```
## Fix: [описание проблемы]

**Root cause:** [почему возникло]
**Fix:** [что изменено]
**Test:** [как проверить что исправлено]
**Prevention:** [как не допустить в будущем]
```
